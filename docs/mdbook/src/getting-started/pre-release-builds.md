# Pre-release builds

We also offer pre-release versions of the observer backend. They are considered less stable though
all our observer builds have a current proud error rate of 0%. It never once had an irrecoverable
issue in the software. 
That said the pre-releases are the ones we are actively working on and have the newest features.
This said you should be aware of issues which can happen. 

Out of the way with the warnings. Here you can install them:

```sh
curl -fsSL https://raw.githubusercontent.com/ObServe-Your-Server/observer/main/setup/deploy-pre.sh | sudo bash
```
