# Mimir

Mimir should become our main storage engine for observer. It can store the data in a binary
format. We will use our own binary format because binstore in rust is no longer maintained. This 
is a bit unfortunate, but I think the learnings and what to do and what not to do will be great.

This probably won't be a perfect implementation by any means, so feel free to get in touch and 
help me become a better dev. Also, quick notice, please no AI.
